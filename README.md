# llama-fastapi
Train and inference llama from scratch
### Requirements
* Linux  
* Docker  
* Nvidia GPU
* Python3
### Install
```
git clone https://github.com/format37/llama-fastapi.git
cd llama-fastapi
sh build.sh
```
### Server start
```
sh run.sh
```
### Call train
```
python3 train.py
```
### Call inference
```
python3 generate.py
```
